## 一  代码管理
版本管理： gitlab + walle
版本控制器： git
开发环境： lnmp  linux + nginx + mysql + php  [reids | memcache | 搜索引擎 + mongoDB]

分支管理：
* master 主分支
* dev 开发人员测试分支
* qa 测试人员测试分支
* uat 灰度分支==发布分支
* release 发布分支，保留版本最近的20个【视情况而定】
* feature 功能分支
* hotfix 修复分支

服务器环境： 推荐 docker
本地环境： 【windos】visual-nmp
